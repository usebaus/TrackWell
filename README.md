# TrackWell

TrackWell is a Flutter-based health tracking app that helps you stay on top of your daily habits, including water intake, exercise, meals, and medications.[cite:18][cite:19]

The app is built as a student capstone project and uses Firebase Authentication and Cloud Firestore to securely store user data.[cite:18][cite:19][cite:26]

---

## Features

- Email and Google login with Firebase Authentication
- Personal dashboard with daily overview
- Water intake tracking with daily goals and progress
- Exercise logging with duration, calories, distance, and intensity
- Simple meal tracking
- Medication list with "taken" status
- Per-user daily goals for water (ml) and exercise (minutes)
- Local Android notifications for daily water and exercise reminders
- Basic settings:
  - Daily water goal
  - Distance unit (km / miles)
  - Reminder times
  - Reset today’s water intake
  - Terms & Conditions note

---

## Tech Stack

- **Framework:** Flutter (Android-focused)
- **Language:** Dart
- **State Management:** Provider (`GoalsProvider` for user goals)
- **Backend:** Firebase
  - Firebase Authentication
  - Cloud Firestore
- **Notifications:** `flutter_local_notifications` with timezone-based scheduling on Android
- **Storage:** Firestore collections per user (`users/{uid}/...`)

## Project Structure (high level)

- `lib/main.dart` – App entry point, Firebase init, routing, Providers
- `lib/dashboard_screen.dart` – Main dashboard with daily summaries and goal progress
- `lib/water_tracker_screen.dart` – Water intake tracking
- `lib/exercise_tracker_screen.dart` – Exercise tracking
- `lib/meal_tracker_screen.dart` – Meal logging
- `lib/medication_tracker_screen.dart` – Medication list and status
- `lib/settings_screen.dart` – Preferences, goals, reminders, T&C note
- `lib/login_screen.dart` & `lib/signup_screen.dart` – Auth flow
- `lib/goal_provider.dart` – Central provider for daily goals
- `lib/notification_service.dart` – Local notification scheduling for Android

## Getting Started (Android)

### Prerequisites

- Flutter SDK installed (3.x)
- Android Studio or VS Code with Flutter plugin
- An Android device or emulator
- A Firebase project configured for Android

### 1. Clone the repository

```bash
git clone https://github.com/usebaus/TrackWell.git
cd TrackWell
```

### 2. Install dependencies

```bash
flutter pub get
```

This will fetch all required packages, including Firebase and `flutter_local_notifications`.[cite:26]

### 3. Configure Firebase (Android)

1. In the Firebase Console, create a project (if you don’t have one).
2. Add an Android app:
   - Use your app’s package name (e.g. `com.example.trackwell` or your actual ID).
   - Download the `google-services.json` file.
3. Place `google-services.json` under:

   ```text
   android/app/google-services.json
   ```

4. Ensure `firebase_options.dart` matches your Firebase project (generated by `flutterfire configure`) or update it accordingly.

### 4. Android notification setup

Add the notification permission to your Android manifest:

```xml
<!-- android/app/src/main/AndroidManifest.xml -->
<manifest ...>
    <uses-permission android:name="android.permission.POST_NOTIFICATIONS" />
    <!-- existing content -->
</manifest>
```

This is required for Android 13+ to show local notifications.[cite:30][cite:36]

The app initializes the notification plugin in `main.dart` and requests permission on startup for Android devices.[cite:18]

### 5. Run on Android

```bash
flutter run
```

Select your Android device or emulator when prompted.

## Usage Overview

- **Sign up:** Create an account with your name, email, and password. You must agree to the Terms & Conditions checkbox.
- **Log in:** Use email/password or Google sign-in.
- **Dashboard:**
  - See today’s water, meals, exercise minutes, and medications.
  - View progress bars for water and exercise with “Goal reached!” indicators.
  - Quick navigation to water, exercise, meal, and medication screens.
- **Water Tracker:** Add intake with quick buttons; see circular and linear progress vs your daily goal.
- **Exercise Tracker:** Log workouts and see daily progress vs your exercise goal.
- **Settings:**
  - Adjust daily water goal (ml).
  - Choose distance unit.
  - Enable reminders and set times for water and exercise.
  - Reset today’s water.
  - Read a short Terms & Conditions note.
